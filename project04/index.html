Project 4: Moroccan Spinner

Description

Introducing the Moroccan spinner. Basically, the spinner has been done by a Moroccan individual, hence its name. Cool thing about it: the motor rotates thanks to a Python program! Moreover, the gear ratio of this spinner is equal to 36:28 x 28:20 = 9:5, which means that input turns 5 times for output to turn 9 times!
