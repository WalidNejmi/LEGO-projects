Project 5: Spirograph

Description

Using a pen, elastics, a LEGO robot, micropython, and my brain, I was able to draw a pretty flower. I am proud that I was able to do this, even as unexperienced as I currently am. Hopefully, one day, I will draw something even better with even more complex machines!

Code

import motor, port, util

# Runs two motors for 10 seconds at different speeds
async def main():
    # Function Parameters: (port, time in milliseconds, speed)
    motor.motor_move_for_time(port.PORTA, 10000, 5000)
    await motor.motor_move_for_time(port.PORTB, 10000, 8000)
    
util.run(main())
