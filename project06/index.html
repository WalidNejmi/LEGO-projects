Project 6: Simple car

Description

I made a car with LEGOs and then programmed the hub to make it move forward, backward, and spin. Below are the videos of the robot moving, along with the code used in Python. PS: I hope when I grow up that I get a better car than this.

Forward Code

import button, motor, port, time

# Causes the program to stall until the center button is pressed
# Do not delete this function!
def wait_until_button_pressed():
    while (button.button_isPressed(button.BUTTON_ON_OFF)[0] == 0):
        time.sleep_ms(1)
    while (button.button_isPressed(button.BUTTON_ON_OFF)[0] == 1):
        time.sleep_ms(1)

# wait for button press
print('Waiting for button press...')
wait_until_button_pressed()

# MOVE FORWARD
# - add your code here for moving the robot forward
motor.motor_move_at_power(port.PORTC, -7000)
motor.motor_move_at_power(port.PORTD, 7000)
time.sleep_ms(2000)
motor.motor_stop()
