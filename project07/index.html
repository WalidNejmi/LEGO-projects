Project 7: Draw a square

Description

Here is an attempt to draw a square using micropython and LEGOs. Although, not perfect, we can see that the drawing looks like a square; the only issue I had was not being able to lift the marker when the motors would make a turn.

Code

# Project 07: Draw a Square
import drivebase, motor, port, time

drivebase.drivebase_init(0, port.PORTC, port.PORTD)

for i in range(4):
    drivebase.drivebase_tank_move_for_time(0, 1750, 1750, 1250)
    time.sleep(2)
    motor.motor_move_at_power(port.PORTB, 1900)
    time.sleep(0.5)
    motor.motor_stop()
    drivebase.drivebase_tank_move_for_degrees(0, 2000, -2000, 235)
    time.sleep(2)
    drivebase.drivebase_stop(0)

drivebase.drivebase_free(0)
