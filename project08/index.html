Project 8: Astronaut Tools

Description

Here are the four videos of the astronaut tool I made to unscrew and screw back nuts. The first video shows me unscrewing a nut without my right hand and the second one with my left hand. The other videos were made to prove that my astronaut tool could work in zero-g. Based on the results, I can proudly say that my tool works in space.

Code

import button, force_sensor, color_sensor, motor, port, sound, time

while (button.button_isPressed(button.BUTTON_ON_OFF)[0] == 0):
    force = force_sensor.get_force(port.PORTA)
    if force == 0:
        motor.motor_move_at_power(port.PORTC, 0)
        print(force)
        time.sleep_ms(1000)
    else:
        if force > 50:
            force = force_sensor.get_force(port.PORTA)
            motor.motor_move_at_power(port.PORTC, force*1000 + 5000)
            time.sleep_ms(1)
        if force < 50:
            force = force_sensor.get_force(port.PORTA)
            motor.motor_move_at_power(port.PORTC, force*-1000 - 5000)
            time.sleep_ms(1)

# BEEP (on end)
sound.beepPlay(500, 100)
