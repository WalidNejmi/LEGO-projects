Project 10: Haunted House

Description

Our project consisted of a door that would open whenever someone pressed the black button. The door would remain open for 3 seconds, accompanied by a screaming voice.

Code

import display, button, force_sensor, color_sensor, motor, port, sound, time
while True:
#while (button.button_isPressed(button.BUTTON_ON_OFF)[0] == 0):
    force = force_sensor.get_force(port.PORTA)
    if force > 0:
        motor.motor_move_at_power(port.PORTE, -5000)
        motor.motor_move_at_power(port.PORTF, 5000)
        print('SCREAM')
        time.sleep_ms(500)
        motor.motor_stop()
        time.sleep_ms(3000)
        motor.motor_move_at_power(port.PORTE, 5000)
        motor.motor_move_at_power(port.PORTF, -5000)
        time.sleep_ms(420)
        motor.motor_stop()
